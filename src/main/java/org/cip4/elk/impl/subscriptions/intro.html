Overview of the Subscriptions package

This package is used to handle the Subscriptions and Persistent Channels in the JDF Specification Release 1.2.

The main class of this package is the {@link org.cip4.elk.impl.subscriptions.AsyncSimpleSubscriptionManager} which implements the functionality of the {@link org.cip4.elk.jmf.SubscriptionManager} interface.

To manage the subscriptions this package include a {@link org.cip4.elk.impl.subscriptions.SubscriptionContainer} which keep track of the {@link org.cip4.elk.impl.subscriptions.Subscription}s that are currently subscribed.

It contains a {@link org.cip4.elk.impl.subscriptions.BaseICSSubscriber} and a {@link org.cip4.elk.impl.subscriptions.BaseICSUnsubscriber} that handles the registering and unregistering of the subscriptions. These are thus classes to reduce the work of the AsyncSimpleSubscritpionManager. The AsyncSimpleSubscritpionManager listens events, keeps track of which Queries are supported, receives Queries and the StopPersChParams and delegates it to the BaseICSSubscriber and the BaseICSUnsubscriber for the registering and unregistering of the Subscritpion.

The package also contains three classes for handling of Time triggered events (The Query/Subscription/@TimeEvent) and an {@link org.cip4.elk.impl.subscriptions.EventForwarder} which is used in integration with Mule to accomplish the asynchronous messaging of subcribed events.
